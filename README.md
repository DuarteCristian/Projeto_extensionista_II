# 🚀 FazTudo

Plataforma web para conectar pessoas que procuram serviços com profissionais que desejam oferecê-los.

## 📌 Funcionalidades

- ✅ Cadastro de usuários
- ✅ Login
- ✅ Perfil do usuário
- ✅ Busca de serviços
- ✅ Oferta de serviços
- ✅ Menu lateral responsivo
- ✅ Interface moderna
- ✅ Navegação entre páginas

## 📸 Telas do Sistema

### Página Inicial
Interface principal da plataforma.

### Login
Autenticação do usuário.

### Cadastro
Registro de novos usuários.

### Perfil
Exibição das informações do usuário.

### Buscar Serviço
Pesquisa por profissionais disponíveis.

### Oferecer Serviço
Cadastro de serviços prestados.

## 🚀 Como Executar

Clone o repositório:

```bash
git clone https://github.com/DuarteCristian/Projeto_extensionista_II
```

Entre na pasta:

```bash
cd faztudo
```

Abra o arquivo:

```bash
index.html
```

ou utilize a extensão **Live Server** no VS Code.
